Como usar no chat:

Markdown: Use **negrito**, *itálico*, `código`, # cabeçalhos, - listas, etc.

Matemática inline: Use $fórmula$ ou símbolos Unicode diretamente

Matemática em bloco: Use $$fórmula$$ para fórmulas grandes
